Markdown is a lightweight markup language used to format text using simple symbols. It lets you write formatted documents (like headings, lists, links, and code) using plain text that is easy to read and write.

It’s widely used in tools like GitHub, Notion, Reddit, and many documentation systems.

Why Markdown is Popular

✅ Very easy to learn
✅ Looks readable even without formatting
✅ Works on many platforms
✅ Great for documentation and notes

Basic Markdown Syntax
1. Headings

Use # symbols.

# Heading 1
## Heading 2
### Heading 3

Result:

Heading 1
Heading 2
Heading 3
2. Bold and Italic
**bold text**
*italic text*

Result:
bold text
italic text

3. Lists

Bullet list

- Item 1
- Item 2
- Item 3

Numbered list

1. First
2. Second
3. Third
4. Links
[OpenAI](https://openai.com)

Result:
OpenAI

5. Images
![alt text](image-url)
6. Code

Inline code:

`print("Hello")`

Code block:

```python
print("Hello world")
```
Where Markdown is Used

Documentation

README files (especially on GitHub)

Note-taking apps like Obsidian and Notion

Forums like Reddit
